# Amazon-Clone
<h3>Amazon Website Clone - Project Description</h3>
The Amazon Website Clone is a front-end web development project designed to replicate the look and feel of the official Amazon e-commerce website. This project focuses on recreating the user inferface, layout of Amazon Website to understand large scale e-commerce platforms are structured.
<br>
The clone emphasises intivtive navigation, and organized product presentation, providing the realistic shopping website experience.
<hr>
<h3>Key Features</h3>
<ul>
  <b>Navigation bar </b>with logo, search box, category menu and cart icon.
  <br>
  <b>Product Listing </b>showing product images, prices, and rating.
  <br>
  <b>Search Bar UI </b>for product searching 
  <br>
  <b>Shoping Cart Section</b>
  <br>
  <b>Footer Section </b>with useful links and policies.
</ul>

</ul>
